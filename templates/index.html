Get CleanSweep

CleanSweep runs directly on your local system to safeguard your privacy and access local caches. Choose your installation method:

Install on macOS (Primary / Recommended) Best for most macOS users. Installs via Homebrew to bypass Gatekeeper:
brew install --HEAD https://raw.githubusercontent.com/DSahir/cleansweep/main/cleansweep.rb
Download for Windows (Core Caches Only) Standalone .exe package. Supported for Windows temporary logs and package caches (Experimental).

For Developers & Advanced Users (All Platforms, including Linux CLI)

git clone https://github.com/DSahir/cleansweep.git
cd cleansweep
python3 app.py

A 3-Step Optimizer Flow

1. Check Live Metrics

Watch system activity load to catch background applications hogging resource headroom.

2. Scan Storage

Analyze directories and breakdown files by categories (Code, Images, Documents).

3. Clean Caches

Scrub cache pools instantly. Safety gates protect your home files and system resources.

Built-in Deletion Safeguards

1. Dry-Run Verification CleanSweep compiles a safe manifest first. No files are deleted until you review and confirm.
2. What is Never Deleted Personal files (Documents, Desktop, Downloads, Pictures), source code projects, and git repos are skipped.
3. Allowlist Path Auditing Every path resolves absolute symlinks using Path.resolve() to prevent parent traversal overrides.
